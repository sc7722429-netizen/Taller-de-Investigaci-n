## Temas Adicionales Implementados

A continuación se describen los 5 temas autónomos investigados que cuentan con ejemplos funcionales en este repositorio:

### 1. Método `isEmpty()`
* **Uso:** Evalúa de forma directa si una cadena de texto tiene longitud 0 o si una colección no contiene elementos.
* **Archivo:** `src/IsEmptyExample.java`

### 2. Método `Math.clamp()` (Java 21)
* **Uso:** Acota de forma matemática cualquier valor numérico dentro de un rango determinado por un mínimo y un máximo.
* **Archivo:** `src/MathClampExample.java`

### 3. Métodos `List.getFirst()` y `List.getLast()` (Java 21)
* **Uso:** Acceso directo y seguro al primer y último elemento de una colección secuencial sin usar índices manuales.
* **Archivo:** `src/SequencedListExample.java`

### 4. Métodos `Files.readString()` y `Files.writeString()` (Java 11)
* **Uso:** Operaciones optimizadas de lectura y escritura de archivos de texto pequeños/medianos en una sola línea.
* **Archivo:** `src/FilesReadWriteExample.java`

### 5. Método `Objects.requireNonNullElse()` (Java 9)
* **Uso:** Validación ágil de nulos que asigna instantáneamente un valor por defecto seguro a una variable.
* **Archivo:** `src/RequireNonNullElseExample.java`
